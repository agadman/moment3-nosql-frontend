Om Webbplatsen

Syfte

Denna webbplats är en del av kursen Backend-baserad webbutveckling hos Mittuniversitetet, våren 2025. Den visar hur man kan använda en webbtjänst för att hantera och presentera arbetserfarenheter.

Hur det fungerar

Webbplatsen använder fetch API för att kommunicera med en Node.js- och Express-baserad backend som är kopplad till en SQL databas. Användaren kan se och lägga till arbetserfarenheter.

Databas

Databasen är en PostgreSQL-databas som körs via Render. Den innehåller tabellen workexperience med följande fält:

  • id
  • companyname
  • jobtitle
  • location
  • startdate
  • enddate
  • description

Slutsatser

Genom detta projekt har jag lärt mig hur frontend och backend kan kommunicera via HTTP och hur man hanterar data i en riktig databas med hjälp av ett REST-API.
